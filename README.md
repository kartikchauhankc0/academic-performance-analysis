# Academic Performance vs Lifestyle Analysis

## 📌 Overview
This project analyzes how students' lifestyle habits like sleep, study time, exercise, and screen time affect academic performance.

## 🚀 Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🔍 Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Machine Learning Model (Random Forest)

## 📊 Insights
- Time management and sleep quality impact performance
- Backlogs negatively affect CGPA
- Study hours alone are not enough

## ▶️ How to Run
1. Install libraries:
   pip install pandas numpy matplotlib seaborn scikit-learn openpyxl

2. Run the project:
   python main.py
